# 空間ボード v4（Supabase版）

畳める空間ボード＋日付タスクのライブ参照＋仮想Inbox。データは Supabase に保存され、ログインすればどの端末（PC・スマホ）からでも同じ状態が見える。タスクの正は Supabase 一本（Todoist 連携は v4 で削除）。

---

## 必要なもの

- Node.js 18 以上
- Supabase アカウント（無料枠で足りる）

---

## セットアップ手順

### 1. Supabase プロジェクトを作る
- https://supabase.com でプロジェクトを新規作成
- リージョンは Tokyo (Northeast Asia) を選ぶと速い

### 2. テーブルを作る
- ダッシュボード左の **SQL Editor** を開く
- 同梱の `supabase-schema.sql` の中身を全部貼り付けて **Run**
- ※ 旧バージョンのテーブルを既に作っている場合は、ファイル末尾のコメントアウトされた ALTER 文を外して実行（box_id の nullable 化が必須）

### 3. 認証を設定する（マジックリンク）
- **Authentication > Providers > Email** が有効なことを確認
- 初回は自分でログインして自分のユーザーを作る（手順6）
- 自分のアカウント作成後、新規サインアップを停止すると他人が登録できない（一人利用なら推奨）

### 4. APIキーを .env に入れる
- **Project Settings > API** から `Project URL` と `anon public` キーをコピー
- `.env.example` を `.env` にリネームして貼る

### 5. 依存をインストール
```
npm install
```

### 6. ローカルで起動
```
npm run dev
```
- 表示された URL を開き、メールでログイン → 動作確認

---

## デプロイ（例：Vercel）

1. このフォルダを GitHub リポジトリに push（`.env` はコミットされない）
2. Vercel でリポジトリを import
3. **Environment Variables** に `VITE_SUPABASE_URL` と `VITE_SUPABASE_ANON_KEY` を登録
4. デプロイ
5. **重要：** Supabase の **Authentication > URL Configuration > Redirect URLs** に、デプロイ先 URL と `http://localhost:5173` の両方を追加（忘れるとログインリンクが機能しない）

## 主な機能

- **プロジェクト箱**（PC＝空間配置 / スマホ＝縦リスト）：タスクに日付・★重要・繰り返し・一行メモ・複数行メモ・URLリンク。
- **付箋**：タスクを持たない自由メモ。
- **仮想 Inbox**：どのプロジェクトにも属さないタスク（box_id なし）が自動で集まる受け皿。Inbox の「＋ 項目」でとりあえず放り込み、後で ⇄ でプロジェクトへ移動。プロジェクトを割り当てた時点で Inbox から自然に消える。
- **タスク移動（⇄）**：どのタスクにも付いている。Inbox⇄プロジェクト、プロジェクト⇄プロジェクトを自由に移動。誤配置は Inbox に戻せる。
- **箱を削除しても中のタスクは消えない**：Inbox へ戻る（DB の on delete set null と一致）。
- **★重要**：サイドパネル最上段に集約（日付セクションとは排他）。
- **サイドパネル**：★重要 / 今日 / 今週 / 今月 / それ以降 / Inbox。日付順。
- **スマホ表示（幅700px以下で自動切替）**：「今やること」（Inbox・★・今日・今週）と「全体」（Inbox＋全プロジェクトのリスト）をトグルで切替。切替状態は端末ごとに記憶。
- **文字サイズ倍率**：A−／A＋（70〜170%）。端末ごとに記憶。

## UI設定の保存場所

文字サイズ倍率・スマホの表示モードは localStorage（端末ごと、同期なし）。タスク・箱・付箋の実体は Supabase（全端末で共通）。

## 既知の制約

- リアルタイム同期は未実装：別端末で開きっぱなしの画面には反映されない。開き直せば最新になる。
- 競合解決は最後の書き込み優先（一人利用なら実害なし）。
- 繰り返しの隔週など interval>1 の曜日指定は近似計算。
- 完了タスクの履歴・アーカイブなし（非表示のみ）。
